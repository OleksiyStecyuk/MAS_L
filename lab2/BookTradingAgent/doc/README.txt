Модифікація до лабораторної роботи №2
До графічного інтерфейсу продавця додана кнопка "Відкрити файл".
Доданий класс HashMap який реалізує функціонал завантаження списку книжок з зовнішнього файлу.
Записи з файлу будуть імпортовані так, як би вони були додані з інтерфейсу продавця.
Формат файлу - txt.
Формат запису в файлі - Назва_книги|Ціна_книги.

Приклад заповненого файлу:
An Introduction To Multiagent Systems|232
Artificial Intelligence: A Modern Approach|153
The State of the Art in Agent Communication Languages|321
