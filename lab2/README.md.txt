Програма модифікована наступним чином:
  - при старті bayer агента в командній стрічці вказується максимальна ціна, яку він готовий заплатити
  - при надсиланні повідомлення до seller агента відправляється request object з назвою книги і максимальною можливою ціною
  - seller агент перевіряє чи є у нього книга по прийнятній ціні і надсилає відповідний результат
